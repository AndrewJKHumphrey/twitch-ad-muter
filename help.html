Overview

Twitch Ad Muter automatically mutes your browser tab when a Twitch ad is playing and unmutes it the moment the ad ends. Multiple detection methods are used so the extension continues to work even when Twitch updates its player.

Additional features include:

  • Mute inactive tabs — automatically mutes Twitch tabs you are not currently watching.
  • Switch tab on ad — switches focus to another Twitch stream when an ad plays on your Primary Tab.
  • Return when ad ends — switches back to your Primary Tab automatically once the ad is over.
  • Close tab when stream ends — closes the tab when a broadcaster ends their stream, with a brief grace period to avoid false closures.
  • Auto-reload on crash — detects stream errors and stalls and reloads the player automatically, up to three times per session.

Settings

Mute inactive tabs Default: off

Mutes every Twitch tab that is not currently in the foreground. Tabs unmute automatically when you switch to them. Useful when you have multiple streams open and only want to hear the one you are actively watching.

Manual mute — if you mute a tab yourself using the browser, the extension will not unmute it. It only unmutes tabs that it muted itself.
Switch tab on ad Default: off

When an ad starts on the Primary Tab, the extension automatically switches focus to another Twitch stream tab in the same window that is not playing an ad. Requires at least two Twitch stream tabs to be open. Non-stream pages (directory, search, home, etc.) are excluded as targets. See Primary Tab below for how to designate one.

Return when ad ends Default: off

Works alongside Switch tab on ad. When the ad finishes on the Primary Tab, the extension switches focus back to it automatically. Interacting with the secondary tab while waiting does not affect this behaviour.

Close tab when stream ends Default: on

Closes the tab automatically when the broadcaster ends their stream. A 5-second grace period is applied before closing to avoid false positives from brief interruptions. If the stream resumes within that window the close is cancelled.

Auto-reload on crash Default: on

Detects stream crashes via Twitch's Error #2000 message and video stall detection (playback frozen for 15+ seconds). When a crash is detected the extension first attempts to click Twitch's own retry button; if none is found it reloads the tab. Limited to 3 reloads per tab within any 10-minute window to prevent infinite reload loops.

Primary Tab

The Primary Tab is the Twitch stream tab you designate as your main viewing tab. It is used by Switch tab on ad and Return when ad ends to know which tab to switch away from when an ad plays and which to return to once the ad is over.

To set a tab as Primary, right-click anywhere on the Twitch page and choose Set as Primary Tab from the context menu. Only one tab can be Primary at a time — setting a new one automatically clears the previous.

Once set, the tab title updates to <streamer> - Primary so you can identify it at a glance in the tab bar. The suffix is maintained even if Twitch updates the title dynamically (e.g. when the stream category or title changes). It is removed automatically when a different tab is made Primary or when the tab is closed.

No automatic assignment — the Primary Tab is never set implicitly by clicking or navigating. You must assign it manually via the right-click menu.

Do Not Mute

Do Not Mute (DNM) prevents the extension from muting a specific tab due to inactivity or any other content-related behaviour. Ads on that tab are still muted — this setting only protects content playback.

To enable it, right-click anywhere on the Twitch page and open the Twitch Ad Muter submenu — the same menu used to set the Primary Tab. Check Do Not Mute This Tab (Ads are still muted) to enable it for that tab. Unchecking it removes the protection.

Per-tab setting — DNM is tracked individually for each tab and is cleared automatically when the tab is closed.
When active, the tab title gains a - DNM suffix so you can identify protected tabs at a glance. If the tab is also the Primary Tab, - Primary takes precedence in the title.

Notes

Selector drift — Twitch periodically changes its HTML class names and data attributes. If muting stops working, check for an extension update.
Tab switching — the extension only considers tabs in the same browser window when looking for an ad-free tab to switch to.
Permissions — the extension requires tabs, storage, and contextMenus permissions. It only operates on twitch.tv pages.
